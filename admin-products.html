Product Manager

Edit product prices, names, discounts, and image paths from one place

How this works

Because this is a static HTML website, browser security will not allow a webpage to directly overwrite project files or commit to GitHub safely. Use this editor to update products, then click Copy Updated Code or Download products-data.js. Replace js/products-data.js, push to GitHub, and Netlify will publish the live website.
